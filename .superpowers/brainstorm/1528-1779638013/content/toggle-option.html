B — Branded "View Intelligence" Button

A more prominent, branded pill button that says "👁 View Intelligence". During demo this is the "wow moment" button — presenter clicks it while explaining the product and the panel dramatically slides in.

✓ Demo-theatrical ✓ Judges see the intent clearly ⚠ Visible to real customers too

Key point: With a toggle button, everything lives at one URL (/chat). The intelligence panel data is already computed and stored in DB after each message — the toggle just shows/hides it. No extra API calls needed when toggling open. Instant reveal.
